模型求解算法

正则化可以有效防止过拟合，增强模型的泛化能力。这时模型的评估策略，就是让结构化的经验风险最小，即增加了正则化项的损失函数最小，称为 结构风险最小化（Structural Risk Minimization，SRM）。

min1n∑i=1nL(yi,f(xi))+λJ(θ)min\frac{1}{n}\sum_{i=1}^{n}L(y_i,f(x_i))+λJ(θ) minn1​i=1∑n​L(yi​,f(xi​))+λJ(θ)

这其实就是求解一个 最优化问题。代入训练集所有数据(xi,yi)(x_i, y_i)(xi​,yi​) ，要求最小值的目标函数就是模型中参数θ\thetaθ的函数。

具体求解的算法，可以利用数学公式直接计算解析解，也可以使用迭代算法。

1 . 解析法

如果模型损失函数的最小值可以通过数学公式进行严格推导，得到一个解析解，那么就直接得到了最优模型的全部参数。这种方法称作解析法。

1.1 特点

  • 适用条件：目标函数必须可导，且导数方程有解析解。
  • 优点：直接且精确；计算高效；
  • 缺点：适用条件较为苛刻；特征维度较大时，矩阵求逆计算复杂度极高。

1.2 应用示例

  • 线性回归问题：可以采用“最小二乘法”求得解析解。

    LossMSE=1n(Xβ−y)T(Xβ−y)Loss_{MSE} = \frac1 n(X\beta - y)^T(X\beta - y) LossMSE​=n1​(Xβ−y)T(Xβ−y)

    ∇LossMSE=2nXT(Xβ−y)=0\nabla Loss_{MSE} = \frac2nX^T(X\beta - y) = 0 ∇LossMSE​=n2​XT(Xβ−y)=0

    β=(XTX)−1XTy\beta = (X^TX)^{-1}X^Ty β=(XTX)−1XTy

可以得到解析解如下：

LossL2=1/n(Xω−y)T(Xω−y)+1nλβTβLoss_{L_2}=1/n (Xω-y)^T (Xω-y)+\frac1n λ\beta^T\beta LossL2​​=1/n(Xω−y)T(Xω−y)+n1​λβTβ

由于加入的对角矩阵 λI 就像一条“山岭”，因此L2正则化也称作“岭回归”。

2. 梯度下降法（重点）

2.1 定义与基本思路

梯度下降法是一种常用的一阶优化方法，用于求解无约束优化问题。其基本思路是先选取一个适当的初始值，然后沿着梯度方向或负梯度方向，不停地更新参数，最终取到极小值。

  • 梯度方向：函数变化增长最快的方向（变量沿此方向变化时函数增长最快）

  • 负梯度方向：函数变化减少最快的方向（变量沿此方向变化时函数减少最快）

由于损失函数是系数的函数，让系数沿着损失函数的负梯度方向变化，损失函数减少最快，能以最快速度下降到极小值，其更新公式为：

θk+1=θk−α⋅∇L(θk)\theta_k+1 = \theta_k - \alpha\cdot\nabla L(\theta_k) θk​+1=θk​−α⋅∇L(θk​)

其中 {▽}{L}\left({θ}_{k}\right)} 是参数取值为 θk{ }{θ}_{k}{ }θk​ 时损失函数 L{L}{ }L 的梯度； α{α }α 是每次迭代的 “步长”，即学习率 。

2.2 特点

  1. 优点：适用性广，能应用于各种不同类型的模型和问题；计算相对简单，不需要复杂的数学推导和计算过程。
  2. 缺点：梯度下降不一定能找到全局最优解，有可能陷入局部最优解；收敛速度较慢，尤其是在处理复杂问题或数据量较大时，需要多次迭代才能接近最优解。

2.3 分类

  1. 批量梯度下降（Batch Gradient Descent，BGD）：每次迭代使用全部训练数据计算梯度，优点是稳定收敛，能保证朝着全局最优解的方向前进；缺点是计算开销大，当训练数据量非常大时，计算梯度的时间和资源成本很高。
  2. 随机梯度下降（Stochastic Gradient Descent，SGD）：每次迭代随机选取一个样本计算梯度，速度快，适合大规模数据，能快速更新参数；但梯度更新方向不稳定，优化过程震荡较大，可能难以收敛到最优解。
  3. 小批量梯度下降（Mini-batch Gradient Descent，MBGD）：每次迭代使用一小批样本（如 32、64 个）计算梯度，平衡了 BGD 的稳定性和 SGD 的速度，是最常用的方法。既可以减少计算量，又能保持一定的稳定性，避免陷入局部最优解。

2.4 计算步骤

  1. 初始化参数：随机选择初始参数，为后续的迭代计算提供起点。
  2. 计算梯度：在当前参数下，计算损失函数的梯度，确定参数更新的方向。
  3. 更新参数：沿负梯度方向调整参数，使损失函数值减小。
  4. 重复迭代：直到满足停止条件（如梯度接近零、达到最大迭代次数等），完成优化过程。

3牛顿法和拟牛顿法（了解）

牛顿法也是求解无约束最优化问题的常用方法，核心思想是利用目标函数的二阶导数信息，通过迭代逐渐逼近极值点。

θ(k+1)=θk−H(−1)(θk)⋅▽L(θk)θ_(k+1)=θ_k- H^(-1) (θ_k)·▽L(θ_k) θ(​k+1)=θk​−H(−1)(θk​)⋅▽L(θk​)

这里的 H−1(θk)H^{-1}(θ_k)H−1(θk​)表示损失函数 L 黑塞矩阵的逆在点θkθ_kθk​ 的取值。

  • 优点：收敛速度快；精度高；
  • 缺点：计算复杂；可能发散。

由于牛顿法中需要计算黑塞矩阵的逆H−1(θk)H^{-1} (θ_k)H−1(θk​)，这一步比较复杂；所以可以考虑用一个n阶正定矩阵来近似代替它，这种方法称为“拟牛顿法”。
牛顿法和拟牛顿法一般用于解决中小规模的凸优化问题。
