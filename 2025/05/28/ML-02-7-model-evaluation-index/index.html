模型评价指标(性能度量)

模型的评价指标用于衡量模型在训练集或测试集上的性能，评估结果反映了模型预测的准确性和泛化能力。

对于回归问题，最常用的性能度量是“均方误差” (Mean Squared Error，MSE)。

1. 回归模型评价指标

1.1 平均绝对误差（MAE）

MAE=1n∑i=1n∣f(xi)−yi∣MAE = \frac 1 n \sum_{i=1}^n |f(x_i) - y_i| MAE=n1​i=1∑n​∣f(xi​)−yi​∣

MAE对异常值不敏感，解释直观。适用于数据包含异常值的场景。

1.2 均方误差（MSE）

MSE=1n∑i=1n(f(xi)−yi)2MSE = \frac 1 n \sum_{i=1}^n (f(x_i) - y_i)^2 MSE=n1​i=1∑n​(f(xi​)−yi​)2

MSE会放大较大误差，对异常值敏感。适用于需要惩罚大误差的场景。

1.3 均方根误差（RMSE）

RMSE=1n∑i=1n(f(xi)−yi)2RMSE = \sqrt{\frac 1 n \sum_{i=1}^n (f(x_i) - y_i)^2} RMSE=n1​i=1∑n​(f(xi​)−yi​)2​

与MSE类似，但量纲与目标变量一致。适用于需要直观误差量纲的场景。如果一味地试图降低RMSE，可能会导致模型对异常值也拟合度也很高，容易过拟合。

1.4 R²（决定系数）

R2=1−∑i=1n(f(xi)−yi)2∑i=1n(yi−yˉ)2R^2 = 1 - \frac {\sum_{i=1}^n(f(x_i) - y_i)^2}{\sum_{i=1}^n(y_i - \bar y)^2} R2=1−∑i=1n​(yi​−yˉ​)2∑i=1n​(f(xi​)−yi​)2​

衡量模型对目标变量的解释能力，越接近1越好，对异常值敏感。

2. 分类模型评价指标

对于分类问题，最常用的指标就是“准确率”（Accuracy），它定义为分类器对测试集正确分类的样本数与总样本数之比。此外还有一系列常用的评价指标。

2.1 混淆矩阵

混淆矩阵（Confusion Matrix）是用于评估分类模型性能的工具，展示了模型预测结果与实际标签的对比情况。

例如，有10个样本。6个是猫，4个是狗。假设以猫为正例，模型预测对了5个猫，2个狗。

2.2 准确率（Accuracy）

正确预测的比例。

Accuracy=TP+TNTP+TN+FP+FNAccuracy = \frac {TP + TN} {TP+TN+FP+FN} Accuracy=TP+TN+FP+FNTP+TN​

上述案例中，准确率=5+210=0.7\frac{5+2}{10}=0.7105+2​=0.7。

2.3 精确率（Precision）

预测为正例的样本中实际为正例的比例，也叫查准率。惩罚项为假正例

Precision=TPTP+FPPrecision=\frac {TP}{TP+FP} Precision=TP+FPTP​

上述案例中，精确率=5/(5+2)=0.7143。

2.4 召回率（Recall）

实际为正类的样本中预测为正类的比例，也叫查全率。惩罚项为假负例

Recall=TPTP+FNRecall=\frac {TP}{TP+FN} Recall=TP+FNTP​

上述案例中，召回率=5/(5+1)=0.8333。

2.5 F1分数（F1 Score）

精确率和召回率的调和平均。

F1Score=2×Precision×RecallPrecision+RecallF1_{Score}=\frac {2×Precision×Recall}{Precision+Recall} F1Score​=Precision+Recall2×Precision×Recall​

上述案例中，F1分数=(2×5/(5+2)×5/(5+1))/(5/(5+2)+5/(5+1))=0.5714。

2.6 报告生成

在代码中，我们可通过sklearn.metrics.classification_report生成分类任务的评估报告，包括精确率、召回率、F1分数等。

  1. Precision（精确率）：也叫查准率，对于每个类别，精确率的计算公式是 Precision=TPTP+FPPrecision = \frac{TP}{TP + FP}Precision=TP+FPTP​，其中 TP（真正例）是被正确预测为正类的样本数，FP（假正例）是被错误预测为正类的样本数。

    在结果中，“猫” 类别的精确率是 0.670.670.67，意味着在预测为 “猫” 的样本中，有 67%67\%67% 确实是猫；“狗” 类别的精确率是 0.710.710.71，即在预测为 “狗” 的样本中，有 71%71\%71% 确实是狗。

  2. Recall（召回率）：也叫查全率，计算公式为 Recall=TPTP+FNRecall = \frac{TP}{TP + FN}Recall=TP+FNTP​，FN（假反例）是被错误预测为负类的样本数。

    “猫” 类别的召回率是 0.500.500.50，表示实际为猫的样本中，有 50%50\%50% 被正确预测出来了；“狗” 类别的召回率是 0.830.830.83，即实际为狗的样本中，有 83%83\%83% 被正确识别出来了。

  3. F1-score（F1 值）：是精确率和召回率的调和平均数，公式为 F1=2×Precision×RecallPrecision+RecallF1 = \frac{2 \times Precision \times Recall}{Precision + Recall}F1=Precision+Recall2×Precision×Recall​。

    “猫” 类别的 F1 值是 0.570.570.57，“狗” 类别的 F1 值是 0.770.770.77。F1 值综合考虑了精确率和召回率，更全面地衡量了分类器在该类别上的性能。

  4. Support（样本数）：表示每个类别的实际样本数量。“猫” 的样本数是 4，“狗” 的样本数是 6。

  5. Accuracy（准确率）：是所有样本中被正确预测的比例，公式为 Accuracy=TP+TNTP+FP+TN+FNAccuracy = \frac{TP + TN}{TP + FP + TN + FN}Accuracy=TP+FP+TN+FNTP+TN​，TN（真反例）是被正确预测为负类的样本数。这里的准确率是 0.700.700.70，即所有 10 个样本中，有 70%70\%70% 被正确分类了。

  6. Macro avg（宏平均）：对每个类别的指标（精确率、召回率、F1 值）进行算术平均。在这个例子中，精确率、召回率和 F1 值的宏平均分别是 0.690.690.69、0.670.670.67 和 0.670.670.67 。宏平均平等地对待每个类别，不考虑类别的样本数量差异。

  7. Weighted avg（加权平均）：根据每个类别的样本数量对指标（精确率、召回率、F1 值）进行加权平均。这里的加权平均精确率、召回率和 F1 值分别是 0.700.700.70、0.700.700.70 和 0.690.690.69 。加权平均考虑了类别的样本数量，样本数量多的类别对结果的影响更大。

2.7 ROC曲线

  • 真正例率（TPR(True positive rate)）：实际为正例，被预测为正例的比例，即召回率。
  • 假正例率（FPR）：实际为负例，被预测为正例的比例。
  • 阈值（Threshold）：根据阈值将概率转换为类别标签。

TPR=TPTP+FNTPR=\frac{TP}{TP+FN} TPR=TP+FNTP​

FPR=FPTN+FPFPR=\frac {FP}{TN+FP} FPR=TN+FPFP​

ROC曲线（Receiver Operating Characteristic Curve，受试者工作特征）是评估二分类模型性能的工具，以假正例率（FPR）为横轴，以真正例率（TPR）为纵轴，展示不同阈值下模型的表现。绘制ROC曲线时，从高到低调整阈值，计算每个阈值的TPR和FPR并绘制所有阈值的点，形成ROC曲线。

2.8 AUC值

AUC值代表ROC曲线下的面积，用于量化模型性能。AUC值越大，模型区分正负类的能力越强，模型性能越好。AUC值=0.5表示模型接近随机猜测，AUC值=1代表完美模型。

可通过sklearn.metrics.roc_auc_score计算AUC值。
