第4章线性回归

1. 什么是线性回归

线性回归（Linear Regression）是一种用于建模两个或多个变量之间线性关系的统计方法。它通过拟合一条直线（或超平面）来描述自变量（输入特征）与因变量（输出目标）之间的关联，并可用于预测或分析变量间的影响关系。

假设因变量y与自变量x1,x2,…,xnx_1,x_2,…,x_nx1​,x2​,…,xn​之间的关系可以用如下线性方程表示：

y=β0+β1x1+β2x2+…+βnxny = \beta_0 + \beta_1x_1 + \beta_2x_2 + … + \beta_nx_n y=β0​+β1​x1​+β2​x2​+…+βn​xn​

  • β0 ：截距，模型在自变量全为0时的基准值

  • β1,β2,…,βn\beta_1,\beta_2,…,\beta_nβ1​,β2​,…,βn​：自变量的系数，表示每个自变量对因变量的影响程度

通过估计这些系数，使模型预测值尽可能接近真实值。

1.1 一元线性回归

仅有一个自变量：y=β0+β1x1y = \beta_0 + \beta_1x_1y=β0​+β1​x1​。

1.2 多元线性回归

包含多个自变量：y=β0+β1x1+β2x2+…+βnxny = \beta_0 + \beta_1x_1 + \beta_2x_2 + … + \beta_nx_ny=β0​+β1​x1​+β2​x2​+…+βn​xn​。

1.3 线性回归应用场景

  • GDP预测：用历史数据（如投资、消费、出口）建立回归模型，预测GDP增长趋势。
  • 广告效果评估：量化不同渠道广告投入对销售额的影响，优化预算分配。
  • 药物剂量研究：分析药物剂量与患者生理指标（如血压、血糖）之间的关系。
  • 产品质量控制：通过生产参数（温度、压力、原料配比）预测产品性能（如强度、耐久）。
  • 政策效果评估：分析最低工资政策对就业率的影响，或环保法规对污染排放的抑制作用。
  • 气候变化建模：用工业排放量、森林覆盖率等变量预测全球气温变化趋势。

2. 线性回归求解(数学推导)

2.1损失函数

模型的预测值与真实值之间存在误差。

损失函数用于衡量模型预测值与真实值之间的误差，并通过最小化损失函数来优化模型参数。当损失函数最小时，得到的自变量系数就是最优解。

1）均方误差（MSE）

MSE=1n∑i=1n(f(xi)−yi)2MSE=\frac1n∑_{i=1}^n(f(x_i)−y_i)^2 MSE=n1​i=1∑n​(f(xi​)−yi​)2

  • n：样本个数。

  • yiy_iyi​：第i个样本的真实值。

  • f(xi)f(x_i)f(xi​)：第i个样本的预测值。

均方误差是回归任务中最常用的损失函数。均方误差对应了欧氏距离。基于均方误差最小化来进行模型求解的方法称为“最小二乘法”。在线性回归中，最小二乘法就是试图找到一条直线（或超平面），使所有样本到直线（或超平面）上的欧氏距离之和最小。

  1. 均方误差的特点

  • 均方误差对大误差比较敏感，因为平方项会放大较大的误差（如误差2→4，误差10→100）。

  • 均方误差是凸函数，存在全局的唯一最小值，平方项又使损失函数处处可导，便于求解最优参数。

  • 最小二乘法（最小化MSE）的解析解可通过矩阵运算直接求出（如β=(XTX)−1XTyβ=(X^T X)^{-1} X^T yβ=(XTX)−1XTy）。

  • 若误差服从正态分布，则均方误差对应极大似然估计，是最优的损失函数。

2）平均绝对误差（MAE）

几何意义

  • 线性回归直线 $ y = \beta_0 + \beta_1 x $ 必然通过点 $ (\bar{x}, \bar{y}) $，即样本均值点。

  • $ \beta_0 $ 是当 $ x = 0 $ 时 $ y $ 的预测值（截距项）。

2.3 正规方程法（解析法）

正规方程法介绍

对于多元线性回归模型 $ y = X\beta + \epsilon $，其中：

  • $ y $ 是 $ n \times 1 $ 的因变量向量，
  • $ X $ 是 $ n \times (p+1) $ 的自变量矩阵（包含常数项列），
  • $ \beta $ 是 $ (p+1) \times 1 $ 的参数向量，
  • $ \epsilon $ 是误差向量。

均方误差损失函数的矩阵形式为：

MSE=1n(y−Xβ)T(y−Xβ)\text{MSE} = \frac{1}{n} (y - X\beta)^T (y - X\beta) MSE=n1​(y−Xβ)T(y−Xβ)

对 $ \beta $ 求偏导数并令其为零：

∂MSE∂β=−2nXT(y−Xβ)=0\frac{\partial \text{MSE}}{\partial \beta} = -\frac{2}{n} X^T (y - X\beta) = 0 ∂β∂MSE​=−n2​XT(y−Xβ)=0

解得正规方程：

XTXβ=XTyX^T X \beta = X^T y XTXβ=XTy

当 $ X^T X $ 为满秩矩阵（即特征之间线性无关）时，可得到参数 $ \beta $ 的解析解：

β=(XTX)−1XTy\beta = (X^T X)^{-1} X^T y β=(XTX)−1XTy

正规方程法的优缺点

  • 优点：

    • 无需迭代，一次性计算出最优解。
    • 当样本量较小且特征维度不高时，计算效率高。

  • 缺点：

    • 当特征数量 $ p $ 很大时，计算 $ (X^T X)^{-1} $ 的复杂度较高（$ O(p^3) $）。
    • 当特征之间存在多重共线性时，$ X^T X $ 接近奇异矩阵，导致数值不稳定。
    • 不适用于大规模数据集（内存限制）。

2.4 梯度下降法

梯度下降的基本原理

梯度下降是一种迭代优化算法，用于最小化目标函数。其核心思想是沿着目标函数的负梯度方向逐步调整参数，直到收敛到局部最小值（对于凸函数，局部最小值即为全局最小值）。

对于损失函数 J(β)J(\beta)J(β)，梯度下降的更新公式为：

βt+1=βt−α⋅∇J(βt)\beta_{t+1} = \beta_t - \alpha \cdot \nabla J(\beta_t) βt+1​=βt​−α⋅∇J(βt​)

其中：

  • βt\beta_tβt​ 是第 t 次迭代的参数值，
  • α\alphaα 是学习率（步长），控制每次迭代的更新幅度，
  • ∇J(βt)\nabla J(\beta_t)∇J(βt​) 是损失函数在 βt\beta_tβt​ 处的梯度。

线性回归中的梯度计算

对于线性回归的均方误差损失函数：

J(β)=12n∑i=1n(hβ(x(i))−y(i))2J(\beta) = \frac{1}{2n} \sum_{i=1}^{n} \left( h_\beta(x^{(i)}) - y^{(i)} \right)^2 J(β)=2n1​i=1∑n​(hβ​(x(i))−y(i))2

其中 hβ(x)=β0+β1x1+⋯+βpxph_\beta(x) = \beta_0 + \beta_1 x_1 + \cdots + \beta_p x_phβ​(x)=β0​+β1​x1​+⋯+βp​xp​ 是预测函数。

∂J(β)∂βj=1n∑i=1n(hβ(x(i))−y(i))xj(i)\frac{\partial J(\beta)}{\partial \beta_j} = \frac{1}{n} \sum_{i=1}^{n} \left( h_\beta(x^{(i)}) - y^{(i)} \right) x_j^{(i)} ∂βj​∂J(β)​=n1​i=1∑n​(hβ​(x(i))−y(i))xj(i)​

参数更新规则为：

βj:=βj−α⋅1n∑i=1n(hβ(x(i))−y(i))xj(i)\beta_j := \beta_j - \alpha \cdot \frac{1}{n} \sum_{i=1}^{n} \left( h_\beta(x^{(i)}) - y^{(i)} \right) x_j^{(i)} βj​:=βj​−α⋅n1​i=1∑n​(hβ​(x(i))−y(i))xj(i)​

（对于 j=0,1,…,pj = 0, 1, \ldots, pj=0,1,…,p，其中 x0(i)=1x_0^{(i)} = 1x0(i)​=1 对应截距项）

学习率的选择

学习率 α\alphaα 的选择至关重要：

  • 学习率过大：可能导致算法发散，无法收敛到最优解。
  • 学习率过小：收敛速度缓慢，需要更多的迭代次数才能达到最优解。

常用策略：

  1. 固定学习率：简单但可能需要多次尝试找到合适的值。
  2. 学习率衰减：随着迭代次数增加逐渐减小学习率。
  3. 自适应学习率算法：如 Adagrad、Adadelta、Adam 等。

梯度下降法常见问题

（1）特征缩放

通常需要提前对特征进行缩放（如标准化或归一化），以加快收敛速度。

（2）局部极小值、鞍点问题

可能陷入局部极小值（非全局最优解），或遇到鞍点（梯度为零但非极值点）。

解决方案：使用动量（Momentum）、自适应优化器（如Adam）或二阶方法（如牛顿法）。

梯度下降的变种

  1. 批量梯度下降（Batch Gradient Descent）：
    • 每次迭代使用全部训练样本计算梯度。
    • 优点：收敛稳定，保证朝着全局最优解方向更新。
    • 缺点：计算开销大，尤其对于大规模数据集。
  2. 随机梯度下降（Stochastic Gradient Descent, SGD）：
    • 每次迭代随机选择一个样本计算梯度。
    • 优点：计算效率高，适用于大规模数据集；引入随机性可能跳出局部极小值。
    • 缺点：收敛过程震荡，可能在最优解附近徘徊，难以确定收敛时机。
  3. 小批量梯度下降（Mini-Batch Gradient Descent）：
    • 每次迭代使用一小批样本（如 32、64、128 等）计算梯度。
    • 结合了批量梯度下降和随机梯度下降的优点，是最常用的实现方式。
