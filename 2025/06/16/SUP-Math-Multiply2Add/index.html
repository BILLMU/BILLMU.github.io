连乘转连加

发表于2025-06-16|更新于2025-06-16|机器学习补充知识数学
|浏览量:

在原文档中，逻辑回归部分涉及多处连乘公式，在对数运算等场景下，连乘可以转换为连加。这是基于对数的运算法则：log⁡(ab)=log⁡a+log⁡b\log(ab)=\log a+\log blog(ab)=loga+logb（a,b>0a,b>0a,b>0），推广到多个数相乘的情况，即log⁡(∏i=1nai)=∑i=1nlog⁡ai\log(\prod_{i = 1}^{n}a_{i})=\sum_{i = 1}^{n}\log a_{i}log(∏i=1n​ai​)=∑i=1n​logai​。以逻辑回归的似然函数为例进行转换说明。

  1. 原似然函数（连乘形式）：对于nnn个样本，逻辑回归的似然函数为L(β)=∏i=1nP(yi∣xi;β)=∏i=1nP(yi=1∣xi;β)yi(1−P(yi=1∣xi;β))1−yiL(β)=\prod_{i = 1}^{n}P(y_i|x_i;β)=\prod_{i = 1}^{n}P(y_i = 1|x_i;β)^{y_i}(1 - P(y_i = 1|x_i;β))^{1 - y_i}L(β)=∏i=1n​P(yi​∣xi​;β)=∏i=1n​P(yi​=1∣xi​;β)yi​(1−P(yi​=1∣xi​;β))1−yi​ 。这里是多个样本的概率乘积形式，在实际优化求解过程中，直接对连乘形式进行计算和求导等操作较为复杂。
  2. 转换为连加形式：对似然函数取对数，得到对数似然函数log⁡L(β)=∑i=1n(yilog⁡P(yi=1∣xi;β)+(1−yi)log⁡(1−P(yi=1∣xi;β)))\log L(β)=\sum_{i = 1}^{n}(y_i\log P(y_i = 1|x_i;β)+(1 - y_i)\log(1 - P(y_i = 1|x_i;β)))logL(β)=∑i=1n​(yi​logP(yi​=1∣xi​;β)+(1−yi​)log(1−P(yi​=1∣xi​;β))) 。通过对数运算，将连乘转换为连加，简化了后续的计算过程。在求解损失函数最小值时，对连加形式的对数似然函数进行求导和优化会更加方便，因为加法的求导规则相对简单，如(u+v)′=u′+v′(u + v)^\prime=u^\prime + v^\prime(u+v)′=u′+v′，相比连乘的求导（如(uv)′=u′v+uv′(uv)^\prime = u^\prime v + uv^\prime(uv)′=u′v+uv′，推广到多个数连乘的求导更为复杂），计算复杂度降低，更易于实现模型的训练和参数优化。

KNN算法 1. KNN算法介绍 K近邻算法（K-Nearest Neighbors，K·NN）是一种基本的分类与回归方法，属于监督学习算法。其核心思想是通过计算给定样本与数据集中所有样本的距离，找到距离最近的K个样本，然后根据这K个样本的类别或值来预测当前样本的类别或值。 1.1 工作原理 计算距离：计算待分类样本与训练集中每个样本的距离。 选择K个近邻：根据计算的距离，选择距离最近的K个样本。 投票或平均： 分类任务：统计K个近邻各类别的数量，将待分类样本归为数量最多的类别。 回归任务：取K个近邻的平均值作为预测结果。 1.2 关键参数 距离度量方法：选择合适的距离度量方法，常见的有欧氏距离、曼哈顿距离、切比雪夫距离、闵可夫斯基距离等。 K值：K值的选择对结果影响很大。K值过小容易过拟合，K值过大则可能欠拟合。 1.3 优缺点 KNN优点： 简单直观，易于理解和实现。 无需训练过程，直接利用训练数据进行预测。 KNN缺点： 计算量大，尤其是训练集较大时。 对噪声数据较敏感。 1.4 API使用 分类 123import numpy as npfrom...
