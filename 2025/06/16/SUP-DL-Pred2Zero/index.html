若要让某个随机变量的期望等于标准正态分布的期望

发表于2025-06-16|更新于2025-06-16|深度学习补充知识
|浏览量:

若要让某个随机变量的期望等于标准正态分布的期望（即 000），并通过常数作为参数来实现这一目标，可通过以下通用方法和示例进行操作。核心思路是：利用常数调整随机变量的位置，使其均值（期望）平移至 000。

一、通用方法：线性变换法

设原始随机变量为 YYY，其期望为 E(Y)=μE(Y) = \muE(Y)=μ。引入常数 ccc，构造新随机变量 X=Y+cX = Y + cX=Y+c，则 XXX 的期望为：

E(X)=E(Y+c)=E(Y)+c=μ+cE(X) = E(Y + c) = E(Y) + c = \mu + c E(X)=E(Y+c)=E(Y)+c=μ+c

若要求 E(X)=0E(X) = 0E(X)=0，则常数 ccc 需满足：

c=−μc = -\mu c=−μ

即通过常数 c=−μc = -\muc=−μ 的平移，使新变量 XXX 的期望为 000。

二、具体分布示例

以下通过不同类型的随机变量（正态分布、均匀分布、离散分布）演示如何通过常数参数使期望为 000。

1. 正态分布情形

原始变量：设 Y∼N(μ,σ2)Y \sim N(\mu, \sigma^2)Y∼N(μ,σ2)，期望 E(Y)=μE(Y) = \muE(Y)=μ，方差 $ \text{Var}(Y) = \sigma^2$。
构造新变量：取常数 c=−μc = -\muc=−μ，定义 X=Y+c=Y−μX = Y + c = Y - \muX=Y+c=Y−μ，则：

X∼N(μ−μ,σ2)=N(0,σ2)X \sim N(\mu - \mu, \sigma^2) = N(0, \sigma^2) X∼N(μ−μ,σ2)=N(0,σ2)

期望验证：E(X)=μ−μ=0E(X) = \mu - \mu = 0E(X)=μ−μ=0。
进一步标准化：若希望 XXX 不仅期望为 000，还满足标准正态分布（方差为 111），可再引入常数 d=1σd = \frac{1}{\sigma}d=σ1​，定义 Z=d⋅X=Y−μσZ = d \cdot X = \frac{Y - \mu}{\sigma}Z=d⋅X=σY−μ​，则：

Z∼N(0,1)Z \sim N(0, 1) Z∼N(0,1)

示例：

  • 若 Y∼N(3,4)Y \sim N(3, 4)Y∼N(3,4)（μ=3\mu = 3μ=3，σ=2\sigma = 2σ=2），取 c=−3c = -3c=−3，则 X=Y−3∼N(0,4)X = Y - 3 \sim N(0, 4)X=Y−3∼N(0,4)，期望为 000；
  • 再取 d=12d = \frac{1}{2}d=21​，则 Z=X2=Y−32∼N(0,1)Z = \frac{X}{2} = \frac{Y - 3}{2} \sim N(0, 1)Z=2X​=2Y−3​∼N(0,1)，同时满足期望为 000 且方差为 111。

2. 均匀分布情形

原始变量：设 Y∼U(a,b)Y \sim U(a, b)Y∼U(a,b)，期望 E(Y)=a+b2E(Y) = \frac{a + b}{2}E(Y)=2a+b​，方差 $ \text{Var}(Y) = \frac{(b - a)^2}{12}$。
构造新变量：取常数 c=−a+b2c = -\frac{a + b}{2}c=−2a+b​，定义 X=Y+c=Y−a+b2X = Y + c = Y - \frac{a + b}{2}X=Y+c=Y−2a+b​，则：

X∼U(a−a+b2,b−a+b2)=U(a−b2,b−a2)X \sim U\left(a - \frac{a + b}{2}, b - \frac{a + b}{2}\right) = U\left(\frac{a - b}{2}, \frac{b - a}{2}\right) X∼U(a−2a+b​,b−2a+b​)=U(2a−b​,2b−a​)

期望验证：E(X)=a+b2−a+b2=0E(X) = \frac{a + b}{2} - \frac{a + b}{2} = 0E(X)=2a+b​−2a+b​=0。
示例：

  • 若 Y∼U(1,5)Y \sim U(1, 5)Y∼U(1,5)（E(Y)=3E(Y) = 3E(Y)=3），取 c=−3c = -3c=−3，则 X=Y−3∼U(−2,2)X = Y - 3 \sim U(-2, 2)X=Y−3∼U(−2,2)，期望为 000。

3. 离散型分布情形

原始变量：设离散随机变量 YYY 取值为 {y1,y2,…,yn}\{y_1, y_2, \dots, y_n\}{y1​,y2​,…,yn​}，概率为 {p1,p2,…,pn}\{p_1, p_2, \dots, p_n\}{p1​,p2​,…,pn​}，期望 E(Y)=∑yipi=μE(Y) = \sum y_i p_i = \muE(Y)=∑yi​pi​=μ。
构造新变量：取常数 c=−μc = -\muc=−μ，定义 X=Y+c=Y−μX = Y + c = Y - \muX=Y+c=Y−μ，则：

E(X)=∑(yi−μ)pi=∑yipi−μ∑pi=μ−μ=0E(X) = \sum (y_i - \mu) p_i = \sum y_i p_i - \mu \sum p_i = \mu - \mu = 0 E(X)=∑(yi​−μ)pi​=∑yi​pi​−μ∑pi​=μ−μ=0

示例：

  • 若 YYY 表示掷一枚公平骰子的点数（取值 1−61-61−6，概率均为 16\frac{1}{6}61​），则 E(Y)=3.5E(Y) = 3.5E(Y)=3.5；
  • 取 c=−3.5c = -3.5c=−3.5，定义 X=Y−3.5X = Y - 3.5X=Y−3.5，则 XXX 的期望为 000，取值为 {−2.5,−1.5,−0.5,0.5,1.5,2.5}\{-2.5, -1.5, -0.5, 0.5, 1.5, 2.5\}{−2.5,−1.5,−0.5,0.5,1.5,2.5}，概率均为 16\frac{1}{6}61​。

三、关键总结

  1. 常数的作用：常数 c=−μc = -\muc=−μ 的本质是将原始变量 YYY 的分布沿数轴平移，使其中心（期望）移至 000，该过程不改变分布的形状（如正态性、均匀性等），仅改变位置。
  2. 与标准正态分布的关联：若原始变量 YYY 是正态分布，通过 c=−μc = -\muc=−μ 和 d=1σd = \frac{1}{\sigma}d=σ1​ 的双重调整（先平移后缩放），可使其成为标准正态分布 N(0,1)N(0, 1)N(0,1)；对于非正态变量，仅能通过平移使期望为 000，但分布形态不变。
  3. 适用场景：
    • 数据标准化：如将考试成绩平移为“以0为中心”的分数（实际中更常用 ZZZ 变换同时标准化期望和方差）；
    • 理论模型构建：在统计学中，有时需构造零均值变量以简化模型假设（如线性回归中的残差期望为0）。
